# React Pizza - Frontend

Фронтенд часть пет-проекта интернет-магазина пиццы

## Возможности:
- Каталог пиццы с фильтрацией и сортировкой
- Корзина
- Личный кабинет:
    - просмотр и редактирование профиля
    - история заказов
- Оформление заказа без авторизации

## Стек:
- React + TS
- Zustand
- Axios + Tanstack Query
- React Hook Form + Zod
- Tailwind CSS
- Vite
